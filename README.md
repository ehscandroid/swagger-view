# Swagger View

A beautiful, interactive OpenAPI/Swagger documentation viewer built with React, TypeScript, and Tailwind CSS.

![Swagger View Demo](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## Features

- **Load specs from anywhere** - Paste a URL, upload a JSON file, or use the built-in demo
- **Interactive API testing** - Execute API requests directly from the documentation
- **Dark mode** - Toggle between light and dark themes
- **Responsive design** - Works seamlessly on desktop and mobile devices
- **Beautiful UI** - Clean, modern interface with emoji support

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

The built files will be in the `dist/` directory. Deploy these to any static hosting service.

## Usage

1. **Load from URL** - Paste an OpenAPI spec URL and click "Load Spec"
2. **Upload file** - Click "Upload file" to select a local `.json` OpenAPI spec
3. **Demo** - Click "Or explore with our demo" to see example documentation

Once loaded, you can:
- Browse endpoints grouped by tags
- Click an endpoint to expand it and see parameters
- Edit parameter values and request body
- Click "Execute" to make a real API call

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS v4
